## Getting Started

**Begin your deep research journey in just a few steps:**

### Prerequisites
- **Python 3.8+:** [Download Python](https://www.python.org/downloads/)
- **pip:** Comes with Python (make sure it‚Äôs up-to-date).

### Installation

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/kvcops/Deep-Research-using-Gemini-api.git
    cd Deep-Research-using-Gemini-api
    ```

2. **Set Up a Virtual Environment (Recommended):**
    ```bash
    python -m venv venv  # Create a virtual environment
    source venv/bin/activate  # Activate on Linux/macOS
    venv\Scripts\activate  # Activate on Windows
    ```

3. **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4. **Configure Your API Key:**
    - Get your **FREE** Google Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey).
    - Create a file named `.env` in the root directory.
    - Add your API key:
      ```env
      GEMINI_API_KEY=YOUR_ACTUAL_GEMINI_API_KEY
      ```

### Running Kv

1. **Start the Application:**
    ```bash
    python app.py
    ```
2. **Open Your Browser:**  
   Go to `http://127.0.0.1:5000/` and start researching!

**Congratulations! Kv is now live and ready to power your deep research.**

## Usage

Using Kv is simple and fun:

1. **Chat Research:**  
   Type your question in the chat box and press Enter. Use "Clear Chat" for a fresh start. You can even upload images for visual help.

2. **Web Search:**  
   Enter a query and click "Web" to get quick, summarized online results with references.

3. **Deep Dive Research:**  
   For a detailed analysis, type your topic and click "Deep Dive" (this might take a little longer).

4. **Customize with Options:**  
   Click "Options" to:
   - Choose Gemini models.
   - Add custom instructions.
   - Select your favorite search engines.
   - Pick your preferred output format.
   - Toggle link/email extraction.

5. **Theme Toggle:**  
   Use the moon/sun icon to switch between light and dark modes for comfortable viewing.
